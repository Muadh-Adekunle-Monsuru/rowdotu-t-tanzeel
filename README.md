# Rowdotu-T-Tanzeel — Landing Page

Under-construction landing page for **Rowdotu-T-Tanzeel Tahfeez & Arabic School**, Ile-Ife, Osun State.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com/new](https://vercel.com/new) and import the repository.
3. Framework preset: **Next.js** (defaults are fine).
4. Deploy.

Images live in `public/images/` (gallery + branding). Source copies remain in `/image` at the project root.
